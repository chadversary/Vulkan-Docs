[[VK_EXT_get_physical_device_format_properties3]]
== VK_EXT_get_physical_device_format_properties3

*Name String*::
    +VK_EXT_get_physical_device_format_properties3+
*Extension Type*::
    Instance extension
*Registered Extension Number*::
    159
*Status*::
    Draft
*Last Modified Date*::
    2017-06-28
*Revision*::
    1.0
*IP Status*::
    No known IP claims.
*Dependencies*::
  - This extension is written against version 1.0.51 of the Vulkan API.
*Contributors*::
  - Chad Versace, Google
*Contacts*::
  - Chad Versace <chadversary@google.com>

The entry point flink:vkGetPhysicalDeviceFormatProperties2KHR has a single
input parameter, pname:format.  This extension provides a new entry point,
flink:vkGetPhysicalDeviceFormatProperties3EXT, that allows additional
parameters to be provided through a ptext:pNext chain.
